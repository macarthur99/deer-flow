# 快速文章总结技能（测试版）

这个技能专门用于快速测试，能高效生成文章摘要和要点列表。

## 核心功能

1. **快速摘要**：生成1-2句总体摘要
2. **要点提取**：创建3-5个关键要点
3. **结构化输出**：使用清晰易读的格式

## 输出格式

**文章摘要**
[1-2句总体概述]

**关键要点**
- [要点1]
- [要点2]
- [要点3]
- [要点4]
- [要点5]

## 工作流程

1. 读取用户提供的文章内容（文件、文本或URL）
2. 快速分析文章核心内容
3. 提取主要观点和关键信息
4. 按照上述模板生成总结
5. 输出结果

## 测试提示

- 保持简单直接
- 专注于核心内容
- 快速响应
- 输出标准化格式

这个技能专为测试设计，确保快速可靠的文章总结功能。
